# Forensic Imaging and Evidence Verification 

 

## Overview 

 

This project demonstrates the process of creating and verifying a forensic image while maintaining the integrity of digital evidence. The investigation focused on acquiring a forensic image, validating hash values, and documenting the process to ensure evidence remained unchanged throughout the examination. 

 

## Objective 

 

The objective of this project was to create a forensic image, verify its integrity using hash values, and understand the importance of evidence preservation during a digital forensic investigation. 

 

## Tools Used 

 

- FTK Imager 

- HashCalc 

- Windows Operating System 

 

## Investigation Process 

 

### Evidence Acquisition 

 

A forensic image was created using FTK Imager. Evidence information was documented during the acquisition process, and the image was saved in RAW (dd) format. 

 

### Hash Verification 

 

Hash values were generated during the imaging process and later verified using HashCalc. Matching hash values confirmed that the forensic image remained unchanged. 

 

### Evidence Preservation 

 

Throughout the project, proper forensic procedures were followed to preserve the integrity and authenticity of the evidence. 

 

## Skills Demonstrated 

 

- Forensic Imaging 

- Hash Verification 

- Evidence Preservation 

- Digital Evidence Handling 

- Documentation 

 

## Key Takeaways 

 

This project reinforced the importance of maintaining evidence integrity and validating forensic images through hash verification. These procedures help ensure that digital evidence remains reliable and admissible throughout an investigation.
